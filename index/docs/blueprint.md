# **App Name**: CampusConnect

## Core Features:

- Authentication: Secure user authentication system with email/password login, JWT-based authentication, and role-based access control (Admin/Student).
- Admin Dashboard: Dashboard for admin users to manage students, rooms, allocations, fees, complaints and notices. Displays key metrics such as total students, available rooms, and pending complaints.
- Student Portal: Portal for students to view their profile, room details, fee status, submit complaints, and view notices.
- Room Allocation: Admin tool for assigning students to rooms and managing room occupancy, integrated with the database.
- Fee Management: System for managing student fees, tracking payments, and generating reports. Accessible to admins and viewable by students.
- Complaint Management: Allows students to submit complaints, which admins can view, resolve, and track.
- Notice Board: Admin tool for posting notices and announcements, visible to all students. Leverage AI to summarise.

## Style Guidelines:

- Primary color: Deep Blue (#3F51B5) to convey trust and reliability.
- Background color: Light gray (#F5F5F5) to ensure readability and a clean interface.
- Accent color: Amber (#FFC107) to highlight important actions and notifications.
- Body and headline font: 'PT Sans', a modern and readable sans-serif for a clean, accessible user experience.
- Use Material Design icons to maintain consistency and provide clear visual cues.
- Responsive layout using Material UI Grid for different screen sizes, ensuring a consistent experience on all devices.
- Subtle transitions and animations to enhance user interaction and provide feedback on actions.
